# react-boilerplate

### Pre-Installs Required
* eslint ```sudo npm i -g eslint```

### Webpack Includes
* HTML, SCSS, React, Babel
* Eslint

### Commands
* npm run start (localhost:8080)
* npm run build (builds in dist folder)

### Entry Files
* **React** (./src/index.js)
* **HTML** (./src/index.html)
* **SCSS** (./src/assets/scss/main.scss)
